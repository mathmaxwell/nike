Nike Landing Page



English

Welcome to the Nike Landing Page, a modern and responsive web page built in December 2024. This project showcases a mobile-friendly design with a custom burger menu, interactive input fields, textareas, and buttons, all crafted using HTML and CSS.

Features





Responsive Design: Adapts seamlessly to small screens and various devices.



Burger Menu: Custom mobile navigation (first-time implementation).



Interactive Elements: Functional input fields, textarea, and buttons.



Clean Code: Well-structured HTML and CSS for easy maintenance.

Installation

To run locally:





Clone the repository:

git clone https://github.com/mathmaxwell/nike.git



Navigate to the project directory:

cd nike



Open index.html in a browser:

open index.html

No dependencies required.

Usage





Resize the browser to test the responsive layout.



Toggle the burger menu on smaller screens.



Interact with input fields, textarea, and buttons.
